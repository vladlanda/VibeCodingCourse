# demo/ — הדגמה חיה, שיעור 2

## מטרת ההדגמה

להראות לכיתה, בזמן אמת, איך פותחים session אמיתי מול כלי Agentic על קוד קיים — באמצעות הפרויקט שנבנה בשיעור 1 (`lesson_01/demo/todo_app`).

## איך להריץ את ההדגמה

1. פתחו VS Code על תיקיית `todo_app` משיעור 1 (או עותק שלה).
2. פתחו את כלי ה-AI האג'נטי (Claude Code / Cursor / Copilot Chat) מול הכיתה.
3. הראו את ההבדל בין Autocomplete (הקלידו קוד לאט, תנו לו להשלים שורה) לבין הנחיה אג'נטית מלאה — ראו `prompt.md`.
4. עצרו על שלב ה-Diff: **תמיד תראו לכיתה איך נראה השינוי המוצע לפני אישור.** זו הנקודה הפדגוגית המרכזית של השיעור.
5. אשרו, הריצו את התוצאה, ועשו Commit מול הכיתה.

## גיבוי

אם הכלי לא זמין (בעיית רשת/הרשאות) — יש להראות צילומי מסך מוכנים מראש של תהליך דומה, ולתאר את השלבים מילולית. אל תדלגו על שלב ה-Diff גם בגיבוי.

זמן ריצה: כ-20-25 דקות (חלק מ-30 הדק' שהוקצו ל-Claude Code בשקפים).
