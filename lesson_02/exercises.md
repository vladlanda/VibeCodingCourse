# תרגול מודרך — שיעור 2 (45 דקות)

## מטרת התרגול

להקים סביבת עבודה אישית שתלווה אתכם לאורך כל הסמסטר, ולבצע בה פעולה אמיתית ראשונה עם כלי AI.

## המשימה

1. **התקינו VS Code** (אם עוד אין).
2. **התקינו Git**, וודאו שיש לכם חשבון GitHub.
3. **בחרו כלי AI חינמי להתחלה** — GitHub Copilot Free הוא ברירת המחדל המומלצת (לא דורש אימות סטודנט). אם כבר יש לכם Student Pack מאושר, אפשר גם דרכו.
4. **צרו Repository חדש וריק בשם `my-vibe-coding-env`** ב-GitHub, ושכפלו אותו (`git clone`) למחשב.
5. **פתחו את התיקייה ב-VS Code**, והפעילו את כלי ה-AI שבחרתם.
6. **בקשו מה-AI ליצור קובץ `hello.py` (או `hello.js`)** שמדפיס הודעת פתיחה אישית — זו הפעולה האג'נטית הראשונה שלכם בקורס.
7. **בדקו את הקוד, הריצו אותו**, ואז בצעו `git add`, `git commit`, `git push`.

## Prompt לדוגמה

```
תיצור קובץ hello.py שמדפיס "שלום מ-Vibe Coding, שיעור 2!" ואת השם שלי
[השם שלכם]. תוסיף גם הדפסה של התאריך הנוכחי.
```

## תוצר מצופה בסוף 45 הדקות (Definition of Done)

- [ ] VS Code מותקן ופתוח על תיקיית הפרויקט.
- [ ] Git מותקן, וה-Repository `my-vibe-coding-env` קיים ב-GitHub.
- [ ] כלי AI חינמי מותקן ופעיל (Copilot Free או שקול).
- [ ] קובץ `hello.py`/`hello.js` נוצר בעזרת AI, נבדק, ורץ בהצלחה.
- [ ] בוצע Commit אחד לפחות ונדחף (Push) ל-GitHub בהצלחה.

## טיפ

אם משהו בהתקנה לא עובד — זה בדיוק סוג הבעיה ש-AI טוב בפתרון שלה. אפשר להעתיק את הודעת השגיאה המדויקת ולשאול את כלי ה-AI "מה השגיאה הזו אומרת ואיך פותרים אותה?" — זה תרגול לגיטימי, לא "רמאות".
