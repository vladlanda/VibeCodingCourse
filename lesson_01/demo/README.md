# demo/ — הדגמה חיה, שיעור 1 (20 דקות)

## מטרת ההדגמה

להראות לכיתה, בזמן אמת, את מלוא ה-Workflow (בגרסה מזורזת) על אפליקציה קטנה — כדי שהתרגול שאחריה (Todo App, 45 דק') לא יהיה הפעם הראשונה שהם רואים את זה בפועל.

## איך להריץ את ההדגמה

1. פתחו כלי AI agentic (Claude Code / Cursor / כלי מקביל) מול הכיתה, מסך משותף.
2. הקלידו את הפרומפט מ-`prompt.md` **בזמן אמת מול הכיתה** — אל תדביקו אותו כבר מוכן, כדי שיראו את התהליך, לא רק את התוצאה.
3. תוך כדי שה-AI עובד — הסבירו לכיתה מה קורה ולמה (זה הזמן להזכיר Hallucination, Context, וכו').
4. אחרי קבלת קוד ראשוני — בדקו אותו מול הכיתה (פתחו בדפדפן), ובקשו שיפור אחד נוסף בזמן אמת כדי להראות איטרציה.
5. אם משהו נכשל — אל תסתירו, זו הזדמנות מצוינת ל"דוגמה חיה לדיבוג" (יורחב בשבוע 9).

## גיבוי

`todo_app/index.html` הוא מימוש-רפרנס מוכן מראש, למקרה שההדגמה החיה נכשלת לגמרי או שנגמר הזמן — אפשר להראות אותו כ"זה מה שהיינו מגיעים אליו", ולא כתחליף להדגמה החיה עצמה.

זמן ריצה: כ-20 דקות. אם מגיעים ל-15 דקות ועדיין באמצע — עברו לסיכום מהיר ("ראיתם את התהליך, בתרגול תעשו את זה בעצמכם עד הסוף").
