# הפרומפט המדויק להקלדה חיה בזמן ההדגמה

הקלידו את זה (או נוסח דומה משלכם) **בזמן אמת** מול הכיתה:

```
בוא נבנה יחד, שלב אחר שלב, אפליקציית Note-Taking פשוטה שרצה בדפדפן —
קובץ HTML יחיד עם CSS ו-JS מוטמעים, בלי שרת.

שלב 1: תתחיל רק במבנה הבסיסי — אזור טקסט לכתיבת פתק חדש, וכפתור שמירה.
אחרי שאני מאשר, נמשיך לשלב הבא (הצגת רשימת הפתקים שנשמרו).
```

לאחר קבלת התוצאה הראשונית, המשיכו בזמן אמת עם:

```
מעולה. עכשיו תוסיף: הצגת כל הפתקים השמורים מתחת לטופס, כל אחד בכרטיס
נפרד, עם כפתור מחיקה. השתמש ב-localStorage לשמירה בין רענוני דף.
```

ואז (אופציונלי, אם יש זמן):

```
תוסיף חיפוש טקסט חופשי שמסנן את הפתקים המוצגים בזמן אמת תוך כדי הקלדה.
```

## למה זו דוגמה טובה להדגמה

- זה **לא** Todo App — כדי שהתרגול (Todo App) לא יהיה "פשוט להעתיק את מה שראינו".
- זה מפורק לשלבים קטנים בכוונה — מדגים "תכנון + מימוש הדרגתי" ולא "תבקש הכל בבת אחת ותקווה".
- localStorage ו-JS פשוט = בלי תלות בהתקנות/שרת, אז לא יהיו בעיות סביבה בזמן ההדגמה.
